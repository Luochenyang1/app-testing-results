| 搜索框   |                           |                | 未点击时，显示“搜索或输入网址”或一个具体的网址；点击后，将有光标在“搜索或输入网址”的文字前闪烁或全选一个具体的网址。输入网址或要搜索的内容，按回车键进入网址或在搜索引擎中显示搜索结果。 |      |

| 取消加载 |                           |                | 点击刷新按钮的时候出现此按钮，点击此按钮则取消加载当前页面。但在常用网站页即使点击刷新按钮也不会出现此按钮。 |      |
| 模式切换 |                           |                | 在桌面、平板、手机三种页面之间切换。                         |      |
| 下载     |                           |                | 点击此按钮，页面显示区显示下载项显示。或在搜索框输入：about:downloads也可达到此效果。 |      |
| 书签     |                           |                | 点击此按钮，则将当前页面加入书签，按钮变中的标志变蓝色，底部弹出书签已添加的提示。再次点击此按钮，则将当前页面移出书签，按钮中的标志变白，底弹出书签已移除的提示。另，点击书签已添加旁边的选项按钮，则弹出对话框，有两个按钮：编辑和添加桌面快捷方式。点击编辑可编辑书签的名称、地址、父文件夹和关键词。点击添加页面快捷方式无效。 |      |
| 标签     |                           |                | 点击则弹出页面显示所有已打开标签                             |      |
